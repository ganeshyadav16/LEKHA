# LEKHA
LEKHA is an innovative web application designed to modernize and streamline the daily operations of local neighborhood grocery stores (Kiranas). By digitizing inventory and store management, it empowers small business owners to operate more efficiently. A standout feature of this platform is its custom integrated voice engine (voice-engine.js), which enables hands-free operations and intuitive voice-based commands, significantly enhancing accessibility for busy shopkeepers. Additionally, the application is built as a Progressive Web App (PWA) utilizing service workers (sw.js) and a web manifest, ensuring reliable offline capabilities and a native app-like experience on mobile devices.

Tech Stack: Frontend: Built entirely with core web technologies including HTML5, CSS3 (with dedicated mobile responsive designs like mobile.css), and Vanilla JavaScript (powering the voice engine, data filtering, and UI interactions).

Backend & Scripting: Utilizes Python (evidenced by scripts like launch_web.py) potentially for local server-side logic or automation, alongside a dedicated backend directory.

Deployment: The application is deployed and hosted on Vercel for fast and reliable web access.
